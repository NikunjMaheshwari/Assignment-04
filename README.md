# Assignment-04: Form Validation Application

## 📌 Project Description

This project is a **Form Validation Application** built using **HTML, CSS, JavaScript, and jQuery**.
It validates user inputs and provides real-time feedback to ensure correct data entry.

---

## 🚀 Live Demo

👉 https://exquisite-bubblegum-4c3a87.netlify.app/

---

## 🎯 Features

* ✅ Required field validation
* ✅ Email format validation
* ✅ Phone number (10 digits) validation
* ✅ Strong password validation
* ✅ Show/Hide password functionality
* ✅ Error messages (red box)
* ✅ Success message (green box)
* ✅ Fully responsive & centered form

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* jQuery
